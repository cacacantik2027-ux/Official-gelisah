## 📋 Tentang Website

Website resmi **GELISAH VCS Talent Agency** — platform untuk menampilkan dan mengelola daftar talent VCS secara profesional. Dibangun dengan HTML, CSS, dan JavaScript murni tanpa framework tambahan, sehingga ringan dan bisa langsung dijalankan di browser manapun.

---

## ✨ Fitur Lengkap

### 🏠 Halaman Utama (Landing Page)
- Tampilan gelap elegan dengan efek gold
- Grid kartu talent dengan animasi hover
- Klik kartu talent → buka halaman profil lengkap

### 👤 Halaman Profil Talent
- Foto, nama, dan deskripsi talent
- Tombol **Order VCS** langsung terhubung ke Telegram bot dengan pesan otomatis sesuai nama talent

### ⚙️ Panel Admin
- Akses via tombol kecil di pojok kanan atas navbar
- **Background** — pilih warna solid atau upload foto dari galeri
- **Musik Latar** — atur link YouTube, aktifkan/nonaktifkan, volume otomatis 50%
- **Talent** — tambah, edit, hapus, ubah urutan tampil
- Upload foto talent langsung dari galeri/perangkat
- Tombol **Simpan Semua** untuk menyimpan sekaligus

### 🎵 Musik Latar Belakang
- Musik YouTube berjalan otomatis di latar belakang
- Tidak ada kontrol musik untuk pengunjung (bersih & rapi)
- Hanya admin yang bisa mengatur musik

### 💬 Live Chat
- Tombol chat di pojok kanan bawah
- Auto-reply otomatis dari GELISAH Support
- Notifikasi muncul otomatis setelah 3 detik

### 📣 Tombol Sosial
- Tombol **GROUP GELISAH** → `t.me/gelisahidpub`
- Tombol **CHANNEL GELISAH** → `t.me/ttalengelisah`

## 🚀 Cara Deploy

### GitHub Pages (Gratis)
1. Upload file `index.html` ke repository ini
2. Buka **Settings** → **Pages**
3. Pilih Branch: `main` → Folder: `/ (root)` → **Save**
4. Website live di: `https://USERNAME.github.io/NAMA-REPO/`

### Netlify Drop (Alternatif, Lebih Cepat)
1. Buka [netlify.com/drop](https://app.netlify.com/drop)
2. Drag & drop file `index.html`
3. Langsung dapat link publik

## 🔧 Cara Pakai Panel Admin

1. Buka website → klik tombol **⚙ Admin** di kanan atas navbar
2. **Atur Background:** pilih warna atau upload foto
3. **Atur Musik:** paste link YouTube, toggle ON/OFF
4. **Kelola Talent:**
   - Klik **+ Tambah Talent** untuk talent baru
   - Isi nama, upload foto dari galeri, isi deskripsi
   - Gunakan tombol ↑ ↓ untuk ubah urutan
   - Klik **Hapus** untuk menghapus talent
5. Klik **💾 Simpan Semua Perubahan**

> ⚠️ Data tersimpan di `localStorage` browser. Untuk hasil terbaik, isi semua data talent di panel admin sebelum link website disebarkan ke pelanggan.
